# Tic Tac Toe Bot

## How to Build and Run

```bash
# Build the project
make

# Run the bot with the following usage:
# ./bin/bot <IP> <PORT> <PLAYER_NUMBER> <PLAYER_NAME> <DEPTH>

# PARAMETERS:
#   IP            - server IP address (e.g., 127.0.0.1)
#   PORT          - server port number (e.g., 8080)
#   PLAYER_NUMBER - player number: 1 (X) or 2 (O)
#   PLAYER_NAME   - player name (max 9 chars)
#   DEPTH         - minimax search depth (1-10)

# EXAMPLE:
./bin/bot 127.0.0.1 8080 1 Cross 5
```
